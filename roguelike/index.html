Become A Rogue

What is this?

"Become A Rogue" is how I felt I needed to create a playable game project, it is mostly to test if I can be a programmer after a brain injury. Essentially to keep myself interested in the project I felt I need to add different features.

In "Become A Rogue" we have the normal roguelike play that you can use, however we are adding several things to this roguelike. You spend your time hunting evil monsters, getting loot (including gear and food), and making an attempt to reach several "end game" bos monsters in the game. It will also have several changes to the project to add start (and weird of funny) things. I wan this game to have more that you can do BESIDES the roguelike dungeon side of the game.

Platform and Library

This is built using Visual Studio 2015 for XNA as such it runs on 32-bit Windows currently.

Depending on how this ends up going (especially since I want to be able to use this on my phone) it will swap to a completely different library (potentially MonoGame) and as many of the platforms as I can pull off.
